README - Cuestionario DSS
=========================

Uso rápido (pragmático)

- Abrir una terminal en la carpeta `parcial3`.
- Ejecutar:

```bash
python3 -m http.server 8000
```

- Abrir en el navegador:

```
http://localhost:8000/prueba01.html
```

- Para usar otro banco de preguntas (archivo JSON en la misma carpeta):

```
http://localhost:8000/prueba01.html?bank=mi_otro_banco.json
```

Alternativa (Node): `npx serve .`

Fin.
